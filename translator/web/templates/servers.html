How to start a server

Run on the machine with the GPU (Mac or Windows):

python server.py --host 0.0.0.0 --port 8765

# Override model path:
python server.py --model-path /path/to/model.gguf

# Disable mDNS (use TCP scan only):
python server.py --no-mdns

# CPU only (no GPU):
python server.py --gpu-layers 0

Then in config.yaml set the translation mode:

remote:
  mode: remote            # always use remote server
  server_url: "http://192.168.1.x:8765"

# or auto-discover:
remote:
  mode: auto              # try LAN first, fall back to local
